# Работа с DevTools: эмуляция мобильного устройства

## Проверка корректного отображения корзины на мобильном устройстве iPhone 12 Pro

### Что проверялось

- Устройство: iPhone 12 Pro
- Разрешение: 390×844
- Страница корзины: https://bugshop.onrender.com/cart

### Результат

- Форма адаптируется под мобильный экран
- Страница соответсвует макету
- При повороте экрана элементы отображаются корректно
- Кнопка "В магазин" доступна для нажатия

### Вывод

Адаптивная вёрстка работает корректно.


# Работа с DevTools: добавление товара в корзину

## Проверка взаимодействия корзины и сервера

### Окружение тестирования

| Параметр | Значение |
|----------|----------|
| Браузер | Google Chrome 146 |
| Инструмент | DevTools → Network |
| URL | https://bugshop.onrender.com/cart |

### Что проверялось

- Отправка запроса при добавлении товара в корзину
- Анализ заголовков запроса (Headers)
- Анализ тела запроса (Payload)
- Анализ ответа сервера (Preview / Response)

### Результаты проверки

| Параметр | Значение |
|----------|----------|
| **Request URL** | https://bugshop.onrender.com/api/cart/items |
| **Request Method** | POST |
| **Status Code** | 200 OK |
| **Payload** | `{"product_id": 1, "quantity": 1}` |
| **Response** | **Сайт является учебным**. |
