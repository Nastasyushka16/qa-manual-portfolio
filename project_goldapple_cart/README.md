# Тестирование корзины интернет-магазина «Золотое Яблоко»

## Цель
Проверка функциональности корзины: добавление/удаление товаров, изменение количества, применение промокодов, расчёт суммы, переход к оформлению.

## Объект тестирования
- **Сайт:** https://goldapple.ru
- **Раздел:** Корзина
- **Окружение:** Yandex Browser, Версия 26.4.2.773 (64-bit), Windows 10 Pro

## Артефакты в этой папке

| Файл | Описание |
| :--- | :--- |
| `test-cases.md` | 26 тест-кейсов (позитивные, негативные, граничные) |
| `bug-reports.md` | 2 баг-репорта (серый «–», невозможность ручного ввода) |
| `checklists.md` | 39 проверок для быстрого регресса |
| `GoldApple.postman_collection.json` | Коллекция Postman |
| `screenshots` | Скриншоты тестирования |

## Что можно улучшить
- Кнопка «–» неактивна если количество товара = 1. - Сделать кнопку активной, чтобы товар можно было удалять.
- Ручной ввод количества невозможен — только кнопками «+» / «–». - Добавить возможность вводить количество вручную, для удобства пользователя.
