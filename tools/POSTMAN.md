| Умение | Что могу делать |
|-------|----------------|
| **Методы HTTP** | GET, POST, PUT, PATCH, DELETE |
| **Окружения** | Настройка и переключение между `test`, `dev`, `prod` |
| **Коллекции** | Создание, структурирование, экспорт и импорт |
| **Переменные** | Разные уровни: глобальные, окружения, коллекции |

## 1. Работа с методами HTTP

- **GET** — получение данных (списка товаров, данных пользователя)
- **POST** — создание (пользователя, добавление товара в корзину)
- **PUT** — обновление (данных пользователя, корзины)
- **PATCH** — частичное обновление (изменение количества товара)
- **DELETE** — удаление (товара из корзины, пользователя)

## 2. Настройка окружений

Переключение между окружениями.

## 3. Переменные

Использовала переменные на разных уровнях:

| **Окружения** | `{{base_url}}` | Базовый URL |
| **Коллекции** | `{{userId}}` | ID пользователя, используется в нескольких запросах |
| **Глобальные** | `{{auth_token}}` | Токен авторизации для всех запросов |

## 4. Тело запроса

Работала с разными форматами:

**JSON (raw)**

{
    "email": "test@gmail.com",
    "password": "qwerty123"
}

## 5. Скрипты (Tests)

проверка статуса:

pm.test("Status code is 200", function () {
    pm.response.to.have.status(200);
});

## 6. Анализ ответа

- **Status Code** — 200, 201, 400, 401, 404, 500
- **Headers** — Content-Type, Authorization
- **Body** — структура JSON, обязательные поля
